## Q&A

#### 1.般若链与PRO币分别是什么？

“般若链”（ProChain）是一个基于以太坊的大数据精准营销区块链，该平台的统一代币称为般若币（PRO）。广告主通过般若链智能合约建立营销任务，推广员完成任务并实时结算奖金。般若链将用户注意力产生的价值归还于用户本身，致力于创造“推广即营销”的病毒传播模式
  
  

#### 2.般若币是否可转给其他人，何时上市交易？
般若币（PRO）符合以太坊ERC20代币标准，可通过以太坊区块链开展交易，我们将在般若币众筹完成后上架主流数字货币交易所。
  
  

#### 3.般若币如何使用？
广告主（Advertiser）可通过参与ICO或公开市场采购般若币（PRO），并将PRO注入智能合约并用于建立营销任务；完成任务的推广员（Promoter）将根据任务价格以及个人传播力获得相应的般若币作为酬劳。

#### 7.般若链的机制是不是传销，有何不同？
不是传销，般若链与传销机制差异巨大：传销组织具有显著的金字塔特性，越高层级的成员可以获得越高比例的分成，成员等级往往高达8~10层。而般若链秉承区块链去中心化特性，用户层级是扁平化的。具体来说，每个账户仅可传播任务至自己的朋友圈，经某个朋友再次传播并完成的任务并不会给原账户带来收益，只会为该朋友带来收益。推般若链的任务传播机制更像是邀请码分发的形式。


#### 8.般若币的众筹计划是怎样的？

般若币总计发行100亿个，ICO众筹时间为期30天，预计发放代币数量60亿个。众筹目标金额为40000ETH，最低筹款额20000ETH，在ICO期间一旦筹满40000ETH则众筹环节直接结束。未达到40000但超过20000，则按照注入ETH总数来计算对应投资者份额，根据份额来分配全部60亿般若币。

在筹满4万ETH的前提下，每1个ETH可兑换150000个PRO；而如果筹到3万ETH，则每个ETH可兑换200000个PRO；如果众筹结束时刚好达到2万ETH最低筹款额，则每个ETH可兑换300000个PRO

#### 11.般若链如何对应广告欺诈？
般若链拥有独有的POSt机制，通过区块链特性对广告欺诈予以彻底的消除。POSt机制是基于账户持币数量与持币时间的一种综合方案，具体来说一个账户在领取任务的时候，将POSt权重作为一个账户是否可领取任务的阈值。POSt具体计算方式为：持币数量×持币秒数。也就是说，某个希望领取任务的账户，需要持有般若币，并且需要保证这部分PRO币在一定时间段内没有交易。POSt机制可有效阻止机器人账号批量刷任务。当某个账户完成特定任务后，该账户下POSt权重将归零并重新计算。而价值越高的任务，对领取任务账号的POSt权重要求也会越高

#### 12.How does the prochain correspond to advertising fraud?

The prochain has a unique POSt mechanism, which is able to eliminated the advertising fraud completely. POSt mechanism a comprehensive program based on the number of account holdings and holding time. Specifically, when a user is going to get a task, the POSt weight will be the threshold to determine whether he can receive it. The POSt calculation method is the number of your PRO multiplied by the seconds you hold the PRO. In other words, you must hold PRO when you hope to get a task. POSt mechanism can effectively prevent the robot accounts to get task. When some user completes a specific task, the POSt weight will be zeroed and recalculated. The higher the value of task is, the higher requirements the account needs.
